# DNA Wallace Tm Calculator

A Python command-line tool to calculate the approximate melting temperature ($T_m$) of short DNA sequences (14–20 base pairs) using the Wallace Rule.

# Method:
The tool utilizes the Wallace formula designed for short oligonucleotides/primers of (max 14-20 base pairs, inaccurate for longer sequences and so most suitable for primers) :
$$T_m = 2(A + T) + 4(G + C)$$
